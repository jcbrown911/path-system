# PATH System

## Overview

PATH (Process & Task Hub) is an AI-assisted workflow and knowledge management framework designed to improve organizational request handling, document management, and cross-department collaboration.

This project began as a graduate AI for Business proposal and continues to evolve as I explore AI implementation, workflow automation, and organizational systems.

---

## Problem

Organizations often rely on emails, conversations, and informal communication to move work forward. This creates delays, duplicated work, and unclear ownership.

---

## Solution

PATH introduces a centralized intake process supported by AI-assisted routing, document management, and a shared knowledge hub.

---

## Technologies

- Microsoft 365
- SharePoint
- OneDrive
- Make.com
- AI-assisted summarization
- Workflow automation
